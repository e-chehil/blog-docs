    市场结构和企业行为¶

    四种基市场结构

    • 完全竞争市场（Perfect Competition）
    • 完全垄断市场（Monopoly）
    • 垄断竞争市场（Monopolistic Competition）
    • 寡头垄断市场（Oligopoly）

    完全竞争市场¶

    概念及特征¶

    完全竞争市场也称为纯粹竞争市场。价格完全由市场决定，企业只是价格的接收者，对定价无能为力。一般具有以下四个假设的市场是属于完全竞争市场：

    1. 市场中有大量的买者与卖者

      每个消费者或厂商只占很小的市场份额，从而他们都是市场价格的接受者，任何一个买者与卖者的行为都不能影响市场价格。

    2. 产品是同质的

      假设所有企业提供的产品是相同的，没有差异的，包括原材料加工、包装、服务、质量等没有差异，所以，企业只靠价格竞争，消费者只凭价格购买。反之，如果商品之间有特色竞争，消费者可能对某一厂商有偏好，就不是完全竞争了。

    3. 生产要素完全自由流动

      在完全竞争下，企业要进入或推出该行业不会受到任何限制，资源将很容易从一种用途转移到另一种用途。没有地域封锁、人为障碍等非价格竞争方式，一切按价格考虑竞争。

    4. 市场信息充分畅通

      企业能够确切了解其销售收入，成本函数，以及生产要素价格各种生产技术等等。消费者也知道各个企业的产品价格，不会因为不了解信息，而上当受骗

    Note

    在现实经济生活中几乎不存在，是一种高度抽象、理想的市场模式，一般作为经济学分析工具，一般认为农产品市场和股票交易市场比较接近完全竞争市场。

    1. 完全竞争的市场结构具有以下四个基本特征：

      1. 买者和卖者很多
      2. 产品同质
      3. 企业进出行业自由
      4. 企业和顾客掌握的市场信息很充分

      Note

      这些基本特征决定了在完全竞争的市场里，产品的价格完全由市场决定，企业只是价格的接受者，对定价无能为力。

    2. 企业短期供给曲线反映从短期看，企业的价格与产量之间的关系。它是在AVC曲线以上的MC曲线部分。

    3. 行业短期供给曲线反映从短期看，在整个行业里价格与产量之间的关系。它等于行业内各企业短期供给曲线的横向相加。
    4. 完全竞争条件下企业的长期均衡的实现条件是：P=MR=MC（短期和长期）= AC（短期和长期）。这时，企业在长期平均成本的最低点生产，经济利润为零，不再有企业进入或离开这个行业。
    5. 成本不变行业的长期供给曲线是一条水平的完全弹性的供给曲线。它表示，从长期看，随着需求的增加，产品价格仍将保持不变，但行业供给量可无限增大。
    6. 成本递增行业的长期供给曲线是一条斜率为正值的供给曲线。它表示，从长期看，产品的价格将会随行业产量的增加而上涨。
    7. 完全垄断是指一个行业只有一家企业，而且它所生产的产品没有其他产品可以替代。
    8. 在完全垄断市场条件下，由于存在市场进入障碍，垄断企业在短期内所获得的经济利润在长期条件下也能够保持下来，这就是垄断利润。
    9. 垄断竞争的市场结构有三个主要特征：
      1. 行业中的企业数量多
      2. 企业进出行业是自由的
      3. 各企业生产的同种产品是有差别的
    10. 垄断竞争条件下企业的长期均衡要求同时满足下列两个条件：
      1. P=AC（短期和长期），此时企业利润为零
      2. MR=MC（短期和长期），此时企业利润最大
    11. 产品差异化竞争就是为自己的产品增加新的、与竞争对手不同的、能更加迎合顾客需要的特征，以吸引更多的消费者。其内容包括：提高产品质量，改进产品性能和结构，增加产品用途，为顾客提供更加周到的服务等。
    12. 广告竞争是产品差异化竞争的重要补充。广告具有提供产品信息，显示乃至强化产品差异的作用。通过广告等促销手段，产品之间的差异往往能更好地为购买者所认识。
    13. 寡头垄断是指几家大企业生产和销售了整个行业的绝大多数产品。
    14. 纯寡头垄断（无差别的寡头垄断）是指在寡头垄断行业中，几家企业生产和销售的产品都是同质的。
    15. 有差别的寡头垄断是指在寡头垄断行业中，几家企业生产和销售的产品在顾客看来是有差别的。
    16. 弯折的需求曲线模式假设：如果一家企业提价，其他企业一般不会跟着提价；但如果一家企业降价，其他企业一般会跟着降价。根据这一假设，寡头垄断企业的需求曲线是一条弯折的需求曲线，相应地，它的边际收入曲线是一条中断的折线。这种模式从理论上说明了为什么在寡头垄断条件下，产品的价格通常具有刚性。
    17. 价格领导模式假设由一家领袖企业首先定价，然后其他企业跟上。领袖企业根据自己的企业需求曲线作出价格和产量决策。在不同价格水平上，领袖企业的需求量等于市场总需求量与其他企业的全部供给量之差。
    18. 卡特尔模式是指几家寡头垄断企业联合起来，共同确定一个价格，以谋求整个行业的总利润最大。
    19. 非价格竞争是指企业利用非价格手段进行竞争，包括产品差异化竞争和促销竞争。
    20. 吓阻进入策略是指现有企业采取策略，人为地为市场设置进入障碍，以阻止新企业的进入。
    21. 有些寡头企业尽管规模较大，但只要新企业进入的沉没成本很低，它的进入和退出就会变得容易。这样，这个行业就依然能保持其竞争性，价格仍有可能保持在长期平均成本最低点附近的水平。这样的市场称为可竞争市场。
    22. 销售收入最大化的企业行为模式假定企业的目标是在保证获得一定利润的条件下，追求销售收入的最大化。对这种企业来说，如果企业的目标利润低于最大化利润，那么企业的价格就应定得比与利润最大化目标对应的价格低，产量则应定得比与利润最大化目标对应的产量高。
