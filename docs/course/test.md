# 基础知识

## 需求

### 需求量

需求量是指在一定时期内，在一定条件下，消费者愿意购买且能够买得起的某种产品或劳务的数量。

### 影响需求量的因素

1.  产品的价格：需求量与商品价格变化成反比（需求法则）

    !!! note "特殊例外的商品"
        1.  吉芬商品：贫困消费者在价格上升时，反而增加对某种低档商品的购买，因为收入效应大于替代效应。
        2.  凡勃伦商品：炫耀性消费，价格越高，反而更能显示消费者的财富和地位，从而增加需求量。

2.  消费者的收入：需求量与消费者的收入变化成正比

3.  相关商品的价格

    - 替代品：两种商品可以互相替代使用，某一商品价格上升，另一种商品的需求量增加。
    - 互补品：两种商品通常一起使用，某一商品价格上升，另一种商品的需求量减少。

4.  消费者的偏好

    厂商的广告营销、流行趋势等因素会影响消费者的偏好，从而影响需求量。

5.  消费者对未来价格的期望

    价格看涨，需求量增加；价格看跌，需求量减少。

### 需求函数

需求函数是需求量与影响这一数量的诸因素之间关系的一种数学表达式，通常表示为：(1)
{ .annotate }

1.  - $Q_x$：商品X的需求量
    - $P_x$：商品X的价格
    - $P_y$：相关商品Y的价格
    - $I$：消费者的收入
    - $T$：消费者的偏好
    - $E$：消费者对未来价格的期望
    - $A$：广告和营销活动

$$Q_x = f(P_x, P_y, I, T, E, A)$$

### 需求曲线

需求曲线是假定在除价格之外其他因素均保持不变的条件下，反映需求量与价格之间关系的表达式，即$Q_D=f(P)$。

- **单个消费者需求曲线**表示单个消费者愿意购买某种产品的数量与其价格之间的关系。
- **行业需求曲线**表示市场上全体消费者愿意购买某种产品的总数与其价格之间的关系，可由行业内所有单个消费者需求曲线的横向加总得到。
- **企业需求曲线**表示某企业的全体顾客愿意向该企业购买某种产品的数量与价格之间的关系，其斜率（绝对值）一般小于行业需求曲线。

!!! note "需求曲线斜率为何向下倾斜？"
    1. 替代效应：当某商品价格下降时，消费者会倾向于购买该商品而减少对其他替代品的购买，从而增加该商品的需求量。
    2. 收入效应：当某商品价格下降时，消费者的实际购买力增加，能够购买更多的该商品，从而增加需求量。

### 需求的变动与需求量的变动

- 需求量的变动：指在其他条件不变的情况下，由于商品价格的变化引起的需求量的变化，表现为沿着需求曲线的移动。
- 需求的变动：指由于影响需求的其他因素变化引起的需求量的变化，表现为需求曲线的整体移动。

## 供给

### 供给量

供给量是指在一定时期内，在一定条件下，生产者愿意且有能力提供某种产品或劳务的数量。

### 影响供给量的因素

1.  本身商品的价格
2.  生产要素的价格
3.  生产技术水平
4.  相关商品的价格
5.  政府税收政策
6.  生产者对价格的预期

## 需求价格弹性

**点价格弹性的计算公式**

$$
\epsilon_{P}=\lim_{\Delta P \to 0} \frac{\Delta Q}{\Delta P} \cdot \frac{P}{Q}=\frac{dQ}{dP} \cdot \frac{P}{Q}
$$

**弧价格弹性的计算公式**

$$
\begin{align*}
E_{P}&=\frac{\Delta Q / Q}{\Delta P / P}=\frac{\left(Q_{2}-Q_{1}\right) / \frac{\left(Q_{1}+Q_{2}\right)}{2}}{\left(P_{2}-P_{1}\right) / \frac{\left(P_{1}+P_{2}\right)}{2}}\\
&=\frac{Q_{2}-Q_{1}}{P_{2}-P_{1}} \cdot \frac{P_{1}+P_{2}}{Q_{1}+Q_{2}}
\end{align*}
$$
