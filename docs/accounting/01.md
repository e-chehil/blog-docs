# 总论

## 会计概述

### 会计的定义

会计是以货币为主要计量单位，反映和监督一个单位经济活动的一种经济管理工作。

### 会计的作用

会计在社会主义市场经济中的作用，主要包括以下几个方面：

1.  提供决策有用的信息，提高企业透明度，规范企业行为。
2.  加强经营管理，提高经济效益，促进企业可持续发展。
3.  考核企业管理层经济责任的履行情况。
